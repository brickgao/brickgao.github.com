POJ-3041 Asteroids

2012-08-30 #Hungary #Graph theory #POJ

题目传送门：POJ 3041

题意为给你一个N*N的矩阵以及所有矩阵中小行星的位置，现在要用炸弹消灭所有的小行星，每个炸弹每次只能炸一行或者一列，求最少用多少炸弹能够消灭所有的小行星。这道题属于最小点覆盖问题。

图论中覆盖的含义是用一些顶点（或边）的集合，使得图中的每一条边（多一个顶点）都至少接触几何中的一个顶点（边）。

最小点覆盖问题就是用最少的点覆盖所有的边。

这里就有一个有趣的定理了......König定理：最小覆盖点数 == 最大匹配数

具体证明可以参照:

http://www.matrix67.com/blog/archives/116

http://blog.sina.com.cn/s/blog_51cea4040100h152.html

消灭每个小行星，就是用炸弹炸掉它的行或者列。我们可以把每个行，每个列分别看成一个点，每个点就相当于两点之间的一条边，所以这样我们就把题目转化为了一个求最小点覆盖的问题。

Code

 1 //By Brickgao
 2 #include <iostream>
 3 #include <cstdio>
 4 #include <cstring>
 5 #include <cmath>
 6 #include <cstdlib>
 7 #include <algorithm>
 8 #include <vector>
 9 #define MAX 10050
10 using namespace std;
11 
12 int n, nn, match;
13 int adjl[MAX][MAX];
14 int mat[MAX];
15 bool used[MAX];
16 
17 bool crosspath(int k)
18 {
19  for(int i = 1; i <= adjl[k][0]; i++)
20  {
21      int j = adjl[k][i];
22      if(!used[j])
23      {
24          used[j] = true;
25          if(mat[j] == 0 || crosspath(mat[j]))
26          {
27              mat[j] = k;
28              return true;
29          }
30      }
31  }
32  return false; 
33 }
34 
35 void hungary()
36 {
37  for(int i = 1; i <= nn; i++)
38  {
39      if(crosspath(i))
40          match++;
41      memset(used, 0, sizeof(used));
42  }
43 }
44 
45 int main()
46 {
47  int a, b;
48  while(scanf("%d%d", &n, &nn) != EOF)
49  {
50      for(int i = 0; i <= n + 1; i++)
51          adjl[i][0] = 0;
52      for(int i = 1; i <= nn; i++)
53      {
54          scanf("%d%d", &a, &b);
55          adjl[a][++adjl[a][0]] = b;
56      }
57      match = 0;
58      hungary();
59      printf("%d\n", match);
60  }
61     return 0;
62 }
